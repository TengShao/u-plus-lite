# U-Plus-Lite — 使用说明书

## 产品简介

U-Plus-Lite 是一个团队月度工作量追踪工具。设计师在每个计费周期内，为每个需求组填报投入的人天数，系统自动换算投入比并给出健康度评估。

---

## 角色说明

| 角色 | 说明 |
|------|------|
| **管理员** | 邵腾（默认账号）。可管理月结周期、成员、需求组、填报工作量 |
| **成员** | 普通设计师。可填报自己参与的需求组的人天投入 |

---

## 快速上手

### 1. 登录

打开应用后使用分配给你的账号登录（联系管理员开通）。

初始管理员账号：`邵腾` / `88888888`

### 2. 选择月结周期

左侧边栏显示所有计费周期，按年分组。点击一个周期，右侧显示该周期下的所有需求组。

### 3. 新建需求组（管理员）

在目标周期下，点击右上角**新建**按钮，输入需求组名称，创建后自动展开编辑。

### 4. 填报工作量

展开任意需求组卡片，在"我的投入"区域：
- 点击骨头（-）和鸡腿（+）按钮调整数值
- 或直接输入数字，步进 0.1 人天
- 点击**提交**保存

### 5. 标记完成（管理员）

需求完成后，点击展开卡片中的绿色**完成**按钮，该需求组标记为"已完成"。

## 界面说明

### 顶部导航栏（71px）
- 左侧：Logo 和标题
- 中间：搜索框，支持按需求组名或设计师姓名搜索
- 右侧：当前用户名，点击展开菜单 → 账户设置 / 退出

### 左侧边栏（320px）
- 按年分组的所有计费周期
- 每个周期显示月份标签、日期范围、OPEN/CLOSED 状态
- 底部"新建月结"按钮（管理员可见）

### 右侧主区域
**操作栏**：筛选下拉框（管线、评级、健康度、设计师）/ 操作按钮

**需求卡片（收起状态）**：
- 需求名称 + 模块/管线/类型标签
- 健康度徽章（绿色=适合 / 黄色=欠饱和 / 红色=过饱和）
- 信息方块：评级、推荐评级、可关闭、人天总数、投入比、参与人数
- 设计师工作量小芯片（显示参与者）

**需求卡片（展开状态）**：
可编辑所有字段：名称、评级、可关闭、模块、管线、类型、预算项
- 设计师列表（只读）
- 我的投入（数字输入）
- 提交 / 完成 / 删除 按钮

## 计算规则

### 级别系数（用于换算人天）

| 级别 | 系数 |
|------|------|
| P3 | 1.0 |
| P4 | 1.3 |
| P5 | 2.0 |
| 外包 | 1.0 |
| 实习 | 0.2 |

**换算人天** = 填报人天 × 级别系数

### 健康度（基于投入比）

| 投入比 | 健康状态 |
|--------|----------|
| 70% ~ 110% | 适合（绿色） |
| < 70% | 欠饱和（黄色） |
| > 110% | 过饱和（红色） |

**投入比** = 换算人天 ÷ 评级标准值 × 100%

| 评级 | 标准人天 |
|------|----------|
| S | 20 |
| A | 10 |
| B | 5 |
| C | 2 |
| D | 0.5 |

## 计费周期规则

- 每月 26 日至次月 25 日为一个周期
- OPEN（开放）：成员可填报数据
- CLOSED（关闭）：只有管理员可编辑

管理员可随时切换周期的开放/关闭状态。

## 管理员功能

点击右上角 ⚙️ 按钮，打开管理面板：

### 成员管理
- 添加/编辑/删除成员
- 设置角色（管理员/成员）
- 设置级别（P5/P4/P3/实习/外包）

### 预算项管理
- 维护管线列表
- 为每个管线添加/编辑/删除预算项

## 常见问题

**Q: 为什么看不到新建按钮？**
A: 只有管理员能在 OPEN 周期下新建需求组。请确认登录的是管理员账号且目标周期状态为 OPEN。

**Q: 提交后数据没保存？**
A: 每个需求组只能有一个未提交的新建需求组，不能同时编辑多个。关闭卡片时会提示未保存的修改。

**Q: 提示版本冲突怎么办？**
A: 说明同时有其他人编辑了同一个需求组。请刷新页面后重新编辑。

**Q: 远程成员如何访问？**
A: 目前需要先连接 VPN 连入办公室网络，再通过局域网 IP 访问。
